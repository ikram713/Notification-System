#  Real-Time Notification System (Admin to User)

This project demonstrates a simple real-time notification system using **Socket.io**. 
It allows an admin to send instant notifications to individual users. The system includes a basic frontend built with **HTML**, **CSS**, and **JavaScript** to test and visualize real-time communication.

---

## Features

- Admin-to-user direct notification
- Real-time updates using  (Socket.io)
- Lightweight frontend interface for testing
- Simple and clean code structure for learning purposes

---

##  Technologies Used

- **Node.js**
- **Express**
- **Socket.io**
- **HTML/CSS/JavaScript**
